# Claude Code Demo (Full)

## Quick Facts

- Stack: React, TypeScript, Node.js
- Test: `npm test`
- Lint: `npm run lint`
- Build: `npm run build`

## Key Directories

- `src/components/` - UI components
- `src/hooks/` - Custom hooks
- `src/api/` - API clients
- `tests/` - Test suites

## Project Rules

- Prefer `interface` over `type` except unions.
- No `any`; use `unknown`.
- Use early returns to reduce nesting.
- Run lint + tests before PR.

## Skill Activation

- Tests -> `testing-patterns`
- GraphQL -> `graphql-schema`
- UI components -> `react-ui-patterns`
